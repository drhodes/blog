Setting up a free blog on github

Published: Sat 27 August 2022
Updated: Sat 27 August 2022
By Derek Rhodes

In how-to.

In this blog post I'll describe how to set up a free math blog on github. We have to start somewhere. That means you are using a bash shell, have python installed and have a github account with ssh-keys configured, you might want to test that out.

10:29 $ ssh -T git@github.com
Hi drhodes! You've successfully authenticated, but GitHub 
does not provide shell access.

Yay! If you get an error instead, then you'll have to debug you ssh configuration, or set it up from scratch.
